#tpmp-lab2-task3

## Overview
Проект для лабораторной работы #2, задание 3.  
Программа на языке C для обработки данных о студентах с использованием структур.  
Реализует ввод данных, сортировку по алфавиту, вычисление среднего балла и фильтрацию студентов.

## Additional Notes
- Программа ожидает ввод данных для 10 студентов (ФИО, курс, 5 оценок).
- Сборка автоматизирована через Makefile.
- В репозитории настроен GitHub Actions для автоматической сборки при push.
- Подключен как submodule в основной репозиторий лабораторной работы.
